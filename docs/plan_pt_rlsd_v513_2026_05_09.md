```python
@register(dispatch_mode=Dispatch.DP_COMPUTE_PROTO)
@DistProfiler.annotate(color="ref")  # R5 compute_ref_log_prob과 동일 annotation
def compute_position_top_k_hits(self, data: DataProto):
    """
    Worker-side top-K check. R5 compute_ref_log_prob의 forward wrap 패턴 그대로:
    - ulysses sequence parallel context
    - FSDP state context manager
    - Properly sharded forward + all-gather logits
    """
    data = data.to(get_device_id())

    micro_batch_size = self.config.ref.log_prob_micro_batch_size_per_gpu
    micro_batch_data = data.batch.split(micro_batch_size)

    in_top_k_chunks = []
    with FSDPState(module=self.ref_module_fsdp), torch.no_grad():
        for batch_td in micro_batch_data:
            with self.ulysses_sharding_manager:
                batch_td = self.ulysses_sharding_manager.preprocess_data(batch_td)
                # forward identical to compute_ref_log_prob shape
                outputs = self.ref_module_fsdp(
                    input_ids=batch_td["input_ids"],
                    attention_mask=batch_td["attention_mask"],
                    position_ids=batch_td.get("position_ids"),
                    use_cache=False,
                )
                logits = outputs.logits  # [B_micro, T, V]
                batch_td = self.ulysses_sharding_manager.postprocess_data(batch_td)

            # Pure logic — extract last valid pos top-K
            attn = batch_td["attention_mask"]
            last_pos = (attn.sum(dim=1).long() - 1).clamp(min=0)
            B = batch_td["input_ids"].size(0)
            last_logits = logits[torch.arange(B, device=logits.device), last_pos]
            top_k = data.meta_info["top_k"]
            target_id = data.meta_info["target_id"]
            top_k_idx = last_logits.topk(top_k, dim=-1).indices
            in_top_k = (top_k_idx == target_id).any(dim=-1).cpu()
            in_top_k_chunks.append(in_top_k)

    in_top_k_full = torch.cat(in_top_k_chunks, dim=0)
    return DataProto.from_dict(tensors={"top_k_hits": in_top_k_full})
```

driver-side caller (verl_sdc.py):
```python
def _compute_position_top_k_signal(trainer, position_batch, meta_start_id, top_k=16):
    """Returns [B] continuous signal in [0, 1]: prob(meta_start) at last valid position.
    NOT binary {0, 1} — continuous to avoid GDPO group-normalize degenerate.
    """
    position_batch.meta_info["top_k"] = int(top_k)
    position_batch.meta_info["target_id"] = int(meta_start_id)

    # ref_in_actor=True (yaml setting): use actor_rollout_wg
    # ref_in_actor=False: use ref_policy_wg
    if trainer.use_reference_policy and trainer.config.actor_rollout_ref.ref.get("ref_in_actor", False):
        wg = trainer.actor_rollout_wg
    else:
        wg = trainer.ref_policy_wg

    out = wg.compute_position_top_k_hits(position_batch)
    return out.batch["top_k_hits"]  # [B] bool, CPU
```

### 3.4 Position penalty 스케일 (revised — continuous signal, CRITICAL 7c fix)

**v5.12 결함**: binary {0, -0.5}는 GDPO group-normalize에서 std=0 (모두 hit / 모두 miss) 시 ε division → advantage 폭주 또는 cancel. R5 step 300 meta emit률 ~95%면 거의 모든 group degenerate.

**v5.13 fix — continuous signal**:

worker는 binary `top_k_hits` 대신 **prob_meta_at_emit_position** 반환:
```python
# in worker compute_position_top_k_hits
# replace binary check:
log_probs_full = torch.log_softmax(last_logits, dim=-1)
prob_meta = torch.exp(log_probs_full[:, target_id])  # [B] in (0, 1]
# Return continuous probability, not binary
return DataProto.from_dict(tensors={"meta_emit_prob": prob_meta.cpu()})
```

driver:
```python
prob_meta = out.batch["meta_emit_prob"]  # [B] continuous
# Position signal = penalty when prob is low (teacher disagrees)
# Use continuous: penalty = -base * (1 - prob_meta) ∈ [-base, 0]
base_penalty = float(config.get("pt_position_penalty_base", -1.0))
```

```python
def _build_position_teacher_batch(
    *,
    tokenizer,
    prompt_texts: list[str],
    gold_answers: list[str],
    responses: torch.Tensor,        # [B, T] response_ids only
    response_mask: torch.Tensor,    # [B, T]
    meta_start_id: int,
):
    """Build teacher input truncated at first META_START position per rollout.

    For rollouts without META emit: skip (no contribution to position_signal).
    For rollouts with META at position p: input = prompt + " Answer: " + gold + " " + decode(response[:p])
    Returns DataProto with [N, T_max] where N = number of rollouts with meta emit.
    Also returns rollout_indices [N] mapping back to original batch positions.
    """
    B = responses.size(0)
    teacher_text_inputs: list[str] = []
    rollout_indices: list[int] = []

    for b in range(B):
        # Find first META_START position in response (response_mask=True)
        valid = (responses[b] == meta_start_id) & response_mask[b].bool()
        nz = valid.nonzero(as_tuple=True)[0]
        if nz.numel() == 0:
            continue  # no meta emitted — skip
        p = int(nz[0].item())
        if p == 0:
            comp_text = ""
        else:
            comp_ids = responses[b, :p]
            comp_mask = response_mask[b, :p].bool()
            comp_ids = comp_ids[comp_mask]
            comp_text = tokenizer.decode(comp_ids, skip_special_tokens=False)
        text = f"{prompt_texts[b]} Answer: {gold_answers[b]} {comp_text}"
        teacher_text_inputs.append(text)
        rollout_indices.append(b)

    if not teacher_text_inputs:
        return None, []  # no meta emit in entire batch — skip position signal entirely

    # Tokenize + pad
    enc = tokenizer(
        teacher_text_inputs,
        add_special_tokens=False,
        padding=True,
        truncation=True,
        max_length=4096,  # conservative; veRL prompt_length + response[:p] should fit
        return_tensors="pt",
    )
    return DataProto.from_dict(
        tensors={"input_ids": enc["input_ids"], "attention_mask": enc["attention_mask"]},
    ), rollout_indices
```

```python
import torch
import numpy as np

def top_k_hits_from_logits(logits, attn_mask, target_id, top_k=16):
    """Pure tensor logic — no verl dependency."""
    last_pos = (attn_mask.sum(dim=1).long() - 1).clamp(min=0)
    B = logits.size(0)
    last_logits = logits[torch.arange(B, device=logits.device), last_pos]
    top_k_idx = last_logits.topk(top_k, dim=-1).indices
    return (top_k_idx == target_id).any(dim=-1)

def prob_meta_at_position(logits, attn_mask, target_id):
    """Continuous signal — return P(target_id) at last valid position."""
    last_pos = (attn_mask.sum(dim=1).long() - 1).clamp(min=0)
    B = logits.size(0)
    last_logits = logits[torch.arange(B, device=logits.device), last_pos]
    log_probs = torch.log_softmax(last_logits, dim=-1)
    return torch.exp(log_probs[:, target_id])

ROD_PT_MODES = {"ROD_PT"}  # 별도 module로 mode set 분리
```

## 9. v5.13 self-critic + round 3 trigger

이번 patch에서 발견된 잠재 결함:

1. **continuous signal scale 검증**: prob_meta가 0.001 ~ 0.99 범위 (R5 step 300 cold start). penalty = -1.0 × (1 - 0.95) = -0.05만큼 약함. `pt_position_penalty_base` 적절 값 ablation 필요. 시작값 -2.0 권고? 또는 logit 직접 사용 (`-base × log_prob_meta` for stronger gradient)?

2. **`_compute_position_top_k_signal`이 worker `compute_position_top_k_hits`를 호출**하는데 worker는 `meta_emit_prob` 반환 (binary가 아닌 continuous). naming inconsistent — worker method 이름도 `compute_position_meta_emit_prob`로 변경 필요.

3. **§3.6 `_build_position_teacher_batch`의 max_length=4096**: prompt + completion[:p] 합본이 4096 초과 가능 (R5 max_response 4096 + prompt 2048 = 6144). truncation이 META_START 위치 잘라낼 위험. left-truncation 또는 prompt-only 우선 처리 필요.

4. **Phase 1a test_no_meta_skip**: 빌더가 None 반환 시 driver가 어떻게 처리하는지 unit test 안 함. driver의 if-None branch 별도 test 필요.
