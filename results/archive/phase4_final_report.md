## Key Findings

### 1. AIME Calibration: Clear improvement path
```
V1 Meta SFT → V2 SFT → V2 E7
    0.870    →  0.712  →  0.610

Δ data: -0.158 (diverse confidence training data)
Δ GRPO: -0.102 (stepwise probe reward)
Total:  -0.260 (30% ECE reduction on hardest benchmark)
```

### 2. Accuracy-Calibration Trade-off
- Best accuracy: Base SFT (58.9%) — no meta, no calibration
- Best AIME ECE: V2 E7 (0.610) — worst accuracy (45.6%)
- Best balance: V1 E5 (55.6% accuracy + 0.071 GSM ECE)
